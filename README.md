## Getting Started

Install this library with your favorite package manager:

```sh
npm i react-aqfa-bar
```

or

```sh
yarn add react-aqfa-bar
```

Then, import and use it:

```sh
import { ProgressBar } from "react-aqfa-bar";

 <ProgressBar
          RaisedValue={50}
          TotalValue={100}
          ContainerStyle={{ width: "500px" }}
        />
```
